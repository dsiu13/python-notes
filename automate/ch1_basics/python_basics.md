# Python Basics

## Operators

| Operator  | Operation | Ex | Output
| :---------: | :-------------: | :---: | :---: |
| ** | Exponent | 2 ** 3 | 8 |
| % | Modulus/remainder | 22 % 8 | 6 |
| // | integer div/floored quotient | 22 // 8 | 2 |
| / | Division | 22 / 8 | 2.75 |
| * | multiplication | 3 * 5 | 15 |
| - | subtraction | 5 - 2 | 3 |
| + | addition | 2 + 2 | 4 |
